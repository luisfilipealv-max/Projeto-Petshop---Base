# PetShop Amigo Fiel
Um projeto web moderno e responsivo para PetShop Amigo Fiel, desenvolvido com foco em boas práticas de desenvolvimento front-end, escalabilidade e experiência do usuário. Ideal para quem busca um ponto de partida sólido para projeto webinstitucionais.


## ✨ Funcionalidades
- **Homepage** com hero section e chamada para ação
- **Seção de serviços** (banho, tosa,  veterinário, etc.)
- **Galeria de imagens**
- **Fotmulário de contato** com validação em javaScript
- **Secão da localização** 
- **Menu responsivo** para dispositivos móveis
- **Design limpo e moderno** com CSS customizado

---

## 🚀 como executar
1. Clone este repositório 
2. Abra a pasta `Projeto-petshop` no seu editor ou navegador
3. Basta abrir o arquivo `index.html` em seu navegador preferido
